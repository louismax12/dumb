1. Smart QR Menu & Self Ordering System (Fokus Pengalaman Pelanggan)

*  Tujuan:
    * Mengurangi waktu tunggu untuk memesan
    * Memberikan informasi menu yang lebih lengkap dan interaktif
    * Memberikan pengalaman yang lebih personal dan modern
* Fitur Utama:
    * Pelanggan memindai QR Code di meja, web aka terbuka menampilkan katalog menu interakti (tanpa perlu install aplikasi). Pelanggan bisa memasukan pesanan ke cart dan langsung checkout (pesanan masuk ke layar dapur/kasir).
    * Pelanggan dapat melihat menu dan memesan langsung dari web, tanpa perlu menunggu pelayan
    * Pelanggan dapat melihat status pesanan dan memantau progres pesanan
    * Pelanggan dapat melihat riwayat pesanan dan memesan ulang
* Nilai Jual ke Pemilik: 
    *  Mengurangi kesalahan catat pesanan
    *  Mempercepat rotasi meja (pelanggan tidak perlu menunggu pelayan datang)
    *  Menu bisa di-update kapan saja melalui panel admin tanpa biaya cetak ulang.
*  Teknis Pembuatan: 
    * Sangat ideal dibangun dengan framework tangguh untuk manajemen database menu dan transaksi, 
    * Dipadukan dengan manipulasi DOM JavaScript untuk kalkulasi cart secara real-time di sisi pelanggan.
